# Project Nightfury

Studio for platformers

## Features

* Player movement (horizontal, vertical, jump)
* Endless platforms ("ground")
* Collision detection
* Camera movement (follow player)
* Melee enemies
* Game over & restarts
* Save & export map
* User profile

## Advanced features

* Leaderboard
* Custom end points
* Texture replacement (replace ground A with ground B)
* Ranged enemies
* Health bars
* Premium stuff
